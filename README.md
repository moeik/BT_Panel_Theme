# BT_Panel_Theme
# 第三方宝塔开源主题

本主题适用于宝塔5.9，其他版本不保证适配


### 说明：

1.修改主题色为黑色<br>
2.简化面板，去除首页软件栏，去除广告推广<br>
3.修改矩形边角为圆角<br>
4.登录界面美化，精简登录页面（超漂亮好吗！）<br>
5.侧边栏图标去除<br>
6.英文化（看起来牛逼一点？）<br>
...<br><br>
学生一枚，能力有限，不喜勿喷~<br>
<br>
### 免责说明：

本主题仅供学习交流使用，请勿用作商业用途！

### 使用方法：
将panel文件夹覆盖到/www/server目录下，重启面板，清除浏览器缓存即可
